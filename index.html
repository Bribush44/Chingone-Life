Quick Export

Google Sheets API Integration

Set up automatic syncing so every food entry is instantly recorded in your Google Sheet. Follow these steps:

1

Create a Google Cloud Project

Go to console.cloud.google.com, create a new project, then enable the Google Sheets API and Google Drive API under APIs & Services → Library.

2

Create OAuth 2.0 Credentials

Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID. Select "Web Application." Add your app's domain to the Authorized JavaScript Origins. Copy the Client ID.

3

Create Your Google Sheet

Create a new Google Sheet. Add this header row in Row 1: Date, User, Meal, Food Name, Quantity, Calories, Protein(g), Carbs(g), Fat(g). Copy the Sheet ID from the URL.

4

Enter Your Credentials Below

Paste your OAuth Client ID and Sheet ID below. Click Connect, then authorize access with your Google account. All future food log entries will sync in real time.

Install on iPhone / iPad
1

Open in Safari

This must be done in Safari — not Chrome or Firefox. If you're reading this in another browser, copy the URL and open it in Safari.

2

Tap the Share button

At the bottom of Safari, tap the Share icon — it looks like a box with an arrow pointing up ⬆️

3

Tap "Add to Home Screen"

Scroll down in the share sheet and tap "Add to Home Screen". The app name will show as Chingone. Tap Add in the top-right corner.

4

Open from your home screen

Find the Chingone icon on your home screen and tap it. It will open fullscreen like a native app — no browser bar, no address bar.

💡 Tip: The app must be hosted at a URL (not opened as a local file) for "Add to Home Screen" to show the custom icon. See hosting options below.
🤖
Install on Android
1

Open in Chrome

Open the app URL in Google Chrome on your Android phone. Chrome has the best PWA support on Android.

2

Tap the install banner or menu

Chrome may automatically show an "Add to Home screen" banner at the bottom. If not, tap the ⋮ three-dot menu in the top-right and select "Add to Home screen" or "Install app".

3

Tap Install and you're done

Confirm the install. The Chingone icon appears on your home screen. It opens fullscreen with no browser UI — just like a real Android app.

How to Host It (Free Options)

The app needs to be at a public URL so your group can access it. Here are the easiest free options:

⚡ Netlify Drop Easiest
Go to app.netlify.com/drop, drag and drop your HTML file onto the page, and you instantly get a public URL like yoursite.netlify.app. Free, no account needed to start.
🐙 GitHub Pages
Create a free GitHub account, make a repository, upload the HTML file as index.html, and enable GitHub Pages in settings. Free forever.
🔥 Firebase Hosting
Google's free hosting. Gives you a clean URL and excellent performance. Requires a Google account and the Firebase CLI tool.
💬
Share with Your Group

Once hosted, share the URL in your WhatsApp group. Anyone who opens it can create an account and install it on their phone. No app store, no download — just open and install.
